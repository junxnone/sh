# 居住证
- 由公安机关签发
- 需要每年签注

## 居住证办理
- **办理地点:** 现居住地的社区事务受理服务中心
- **条件**: 
  - 外地户口
  - 办理居住登记满半年

## 居住证签注
- 《居住证》每届满1年前30日内，到社区事务受理服务中心办理签注手续
